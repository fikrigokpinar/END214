NT–01
Üstel Dağılım — EÇO Tahmini ve Yansızlık
EÇO · Üstel Temel
▾
X₁, X₂, …, Xₙ b.ö.d. rasgele değişkenler, parametresi θ > 0 olan üstel dağılımdan gelmektedir:

f(x; θ) = (1/θ) e−x/θ,   x > 0

Burada θ = E(X) ortalama parametresidir.
a)θ parametresinin en çok olabilirlik tahmin edicisini (EÇO) türetiniz.
b)EÇO tahmin edicisinin yansız olup olmadığını gösteriniz.
Adım Adım Çözüm
Olabilirlik
L(θ) = ∏ni=1 f(xᵢ; θ) = ∏ni=1 (1/θ) e−xᵢ/θ
= (1/θn) · e−(1/θ)∑xᵢ
Log-olabilirlik
ℓ(θ) = ln L(θ) = −n lnθ − (1/θ)∑xᵢ
Türev & çözüm
dℓ/dθ = −n/θ + ∑xᵢ/θ² = 0
θ · n = ∑xᵢ  →  θ̂ = ∑Xᵢ/n = X̄
d²ℓ/dθ² = n/θ² − 2∑xᵢ/θ³; θ̂'de < 0 → maksimum ✓
b — Yansızlık
E(θ̂) = E(X̄) = E(X₁) = θ  [üstel dağılımda E(X) = θ]
→ θ̂ = X̄ yansızdır.
NT–02
Poisson Dağılımı — EÇO Tahmini ve Yansızlık
EÇO · Poisson Temel
▾
X₁, …, Xₙ b.ö.d. Poisson(λ) dağılımından gelmektedir:

P(X = x; λ) = e−λ λx / x!,   x = 0,1,2,…
a)λ parametresinin EÇO'sunu türetiniz.
b)EÇO tahmin edicisinin yansız olduğunu gösteriniz.
Adım Adım Çözüm
Log-olas.
ℓ(λ) = ∑[xᵢ lnλ − λ − ln(xᵢ!)]
= (∑xᵢ) lnλ − nλ − sabit
Türev
dℓ/dλ = ∑xᵢ/λ − n = 0  →  λ̂ = X̄ = ∑Xᵢ/n
b
E(λ̂) = E(X̄) = E(X₁) = λ  [Poisson'da E(X) = λ]
→ X̄ yansızdır.
NT–03
Düzgün Dağılım — Momentler Yöntemi
Momentler · Düzgün Temel
▾
X₁, …, Xₙ b.ö.d. Düzgün(0, θ) dağılımından gelmektedir:

f(x; θ) = 1/θ,   0 < x < θ
a)Momentler yöntemiyle θ için bir tahmin edici türetiniz.
b)Bu tahmin edicinin yansız olup olmadığını gösteriniz.
Adım Adım Çözüm
a — 1. moment
μ₁ = E(X) = θ/2
Örneklem 1. momenti: m₁ = X̄
θ/2 = X̄  →  θ̂ᴩ = 2X̄
b
E(2X̄) = 2·E(X̄) = 2·θ/2 = θ
→ 2X̄ yansızdır.
NT–04
Normal Dağılım — EÇO ile μ ve σ² Tahmini
EÇO · Normal · İki Parametre Temel
▾
X₁, …, Xₙ b.ö.d. N(μ, σ²) dağılımından gelmektedir:

f(x; μ, σ²) = (1/√(2πσ²)) exp[−(x−μ)²/(2σ²)]
a)μ ve σ² için EÇO tahmin edicilerini türetiniz.
b)σ̂² = ∑(Xᵢ−X̄)²/n tahmin edicisinin yansız olup olmadığını gösteriniz. Yansız hale getiriniz.
Adım Adım Çözüm
Olabilirlik
L(μ,σ²) = ∏ni=1 (2πσ²)−1/2 exp[−(xᵢ−μ)²/(2σ²)]
= (2πσ²)−n/2 exp[−∑(xᵢ−μ)²/(2σ²)]
Log-olabilirlik
ℓ(μ,σ²) = −n/2 · ln(2π) − n/2 · ln(σ²) − ∑(xᵢ−μ)²/(2σ²)
a — ∂/∂μ
∂ℓ/∂μ = −∑ 2(xᵢ−μ)·(−1)/(2σ²) = ∑(xᵢ−μ)/σ² = 0
∑xᵢ − nμ = 0  →  μ̂ = X̄ = ∑Xᵢ/n
a — ∂/∂σ²
∂ℓ/∂σ² = −n/(2σ²) + ∑(xᵢ−μ)²/(2σ⁴) = 0
∑(xᵢ−μ̂)² = nσ²  →  σ̂² = ∑(Xᵢ−X̄)²/n
b — Açılım
∑(Xᵢ−X̄)² = ∑[Xᵢ²−2XᵢX̄+X̄²] = ∑Xᵢ² − 2X̄·nX̄ + nX̄² = ∑Xᵢ² − nX̄²
b — E(Xᵢ²)
E(Xᵢ²) = Var(Xᵢ) + [E(Xᵢ)]² = σ² + μ²
b — E(X̄²)
E(X̄²) = Var(X̄) + [E(X̄)]² = σ²/n + μ²
b — E[∑(Xᵢ−X̄)²]
E[∑Xᵢ²] − nE[X̄²]
= n(σ²+μ²) − n(σ²/n+μ²)
= nσ²+nμ² − σ² − nμ²
= (n−1)σ²
b — Sonuç
E(σ̂²) = E[∑(Xᵢ−X̄)²] / n = (n−1)σ²/n ≠ σ²
→ σ̂² yansız değildir; (n−1)/n kadar küçük tahmin eder.
Yansız hale getirmek için paydayı (n−1) yaparız:
S² = ∑(Xᵢ−X̄)²/(n−1)  →  E(S²) = σ² ✓
NT–05
Düzgün Dağılım — EÇO ve Yansızlık Düzeltmesi
EÇO · Düzgün · Yansızlık Orta
▾
X₁, …, Xₙ b.ö.d. Düzgün(0, θ) dağılımından gelmektedir:

f(x; θ) = 1/θ,   0 < x < θ
a)θ parametresinin EÇO'sunu türetiniz. (İpucu: olabilirlik fonksiyonu türev yöntemiyle değil, tanım bölgesi kısıtını göz önüne alarak maksimize edilir.)
b)EÇO tahmin edicisinin yansız olmadığını gösteriniz ve yansız versiyonunu bulunuz.
Adım Adım Çözüm
Olabilirlik
L(θ) = ∏ni=1 f(xᵢ;θ) = ∏ni=1 (1/θ) = (1/θ)n
Ancak bu ifade yalnızca tüm xᵢ ≤ θ olduğunda geçerlidir; aksi halde L(θ) = 0.
Yani: L(θ) = (1/θ)n   eğer   θ ≥ max(xᵢ) = x(n)
a — Maksimum
L(θ) = (1/θ)n, θ ≥ x(n) koşuluyla.
θ arttıkça (1/θ)n monoton azalır. Dolayısıyla L(θ), θ'nın izin verilen en küçük değeri olan x(n)'de maksimum olur.
EÇO: θ̂ = X(n) = max(X₁, …, Xₙ)
b — X(n)'nin dağılımı
n. sıra istatistiğinin yoğunluk fonksiyonu:
f(n)(x) = n [F(x)]n−1 f(x) = n (x/θ)n−1 · (1/θ) = nxn−1/θn,   0 < x < θ
b — E(X(n))
E(X(n)) = ∫0θ x · nxn−1/θn dx = (n/θn) ∫0θ xn dx
= (n/θn) · [xn+1/(n+1)]0θ = (n/θn) · θn+1/(n+1)
E(X(n)) = nθ/(n+1) ≠ θ  →  yansız değil.
b — Yansız düzeltme
T = c · X(n) yansız olsun: E(T) = c · nθ/(n+1) = θ  →  c = (n+1)/n
Yansız tahmin edici: T = [(n+1)/n] · X(n)
Doğrulama: E(T) = (n+1)/n · nθ/(n+1) = θ ✓
NT–06
Genel Kuvvet Dağılımı — Momentler ve EÇO Karşılaştırması
Momentler · EÇO · Karşılaştırma Orta
▾
X₁, …, Xₙ b.ö.d. aşağıdaki dağılımdan gelmektedir:

f(x; θ) = (θ+1) xθ,   0 < x < 1,   θ > −1
a)Momentler yöntemiyle θ için bir tahmin edici bulunuz.
b)EÇO yöntemiyle θ için bir tahmin edici bulunuz.
Adım Adım Çözüm
a — 1. moment
μ₁ = E(X) = ∫01 x·(θ+1)xθdx = (θ+1)·[xθ+2/(θ+2)]01 = (θ+1)/(θ+2)
m₁ = X̄ eşitle: (θ+1)/(θ+2) = X̄
θ+1 = X̄(θ+2)  →  θ(1−X̄) = 2X̄−1
θ̂ᴩ = (2X̄−1)/(1−X̄)
b — Olabilirlik
L(θ) = ∏ni=1 (θ+1)xᵢθ = (θ+1)n ∏xᵢθ
b — Log-olabilirlik
ℓ(θ) = n ln(θ+1) + θ∑ln(xᵢ)
b — Türev & çözüm
dℓ/dθ = n/(θ+1) + ∑ln(xᵢ) = 0
θ̂ᴥ = −1 − n/∑ln(Xᵢ) = −1 − 1/̅(ln X)
NT–07
Üstel Dağılım — İki Yöntemin Karşılaştırması ve Yansızlık
EÇO · Momentler · Yansızlık Orta
▾
X₁, …, Xₙ b.ö.d. μ ortalama parametreli üstel dağılımdan gelmektedir:

f(x; μ) = (1/μ) e−x/μ,   x > 0,   μ > 0
a)Momentler yöntemiyle μ için bir tahmin edici bulunuz ve yansızlığını gösteriniz.
b)EÇO yöntemiyle μ için bir tahmin edici bulunuz. İki yöntem aynı sonucu verir mi?
c)X̄'nın tutarlı tahmin edici olduğunu argümanlayınız.
Adım Adım Çözüm
a — Momentler
μ₁ = E(X) = μ  →  m₁ = X̄ eşitle: μ̂ᴩ = X̄
E(X̄) = μ → yansızdır ✓
b — Olabilirlik
L(μ) = ∏ni=1 (1/μ) e−xᵢ/μ = (1/μn) e−(1/μ)∑xᵢ
b — Log-olabilirlik
ℓ(μ) = −n lnμ − (1/μ)∑xᵢ
b — Türev & çözüm
dℓ/dμ = −n/μ + ∑xᵢ/μ² = 0  →  μ = ∑xᵢ/n  →  μ̂ᴥ = X̄
Her iki yöntem aynı sonucu vermektedir.
c — Tutarlılık
Büyük Sayılar Yasası gereği X̄ →p E(X) = μ ve Var(X̄) = μ²/n → 0.
→ X̄ tutarlı tahmin edicidir.
NT–08
İki Parametreli Düzgün Dağılım — Momentler Yöntemi
Momentler · Düzgün · İki Parametre Orta
▾
X₁, …, Xₙ b.ö.d. Düzgün(α, β) dağılımından gelmektedir:

f(x; α, β) = 1/(β−α),   α < x < β
a)Momentler yöntemiyle α ve β için tahmin edicileri türetiniz.
Adım Adım Çözüm
Teorik momentler
μ₁ = E(X) = (α+β)/2
μ₂ = E(X²) = (α²+αβ+β²)/3
Var(X) = μ₂−μ₁² = (β−α)²/12
Örneklem momentleri
m₁ = X̄   |   Var tahmini: S²n = ∑(Xᵢ−X̄)²/n
Denklem sistemi
1. moment: (α+β)/2 = X̄  →  α+β = 2X̄  …(1)
Varyans: (β−α)²/12 = S²n  →  β−α = 2√(3S²n)  …(2)
Çözüm
(1) + (2): 2β = 2X̄ + 2√(3S²n)  →  β̂ = X̄ + √(3S²n)
(1) − (2): 2α = 2X̄ − 2√(3S²n)  →  α̂ = X̄ − √(3S²n)
NT–09
Etkinlik Karşılaştırması — İki Yansız Tahmin Edici
Etkinlik · Karşılaştırma Orta
▾
X₁, …, Xₙ b.ö.d. N(μ, σ²) dağılımından gelmektedir. Aşağıdaki iki tahmin edici önerilmektedir:

T₁ = X̄   |   T₂ = (X₁ + Xₙ) / 2
a)Her iki tahmin edicinin yansız olduğunu gösteriniz.
b)Varyanslılıklarını karşılaştırınız. Hangisi daha etkindir ve neden?
Adım Adım Çözüm
a
E(T₁) = E(X̄) = μ ✓
E(T₂) = (E(X₁)+E(Xₙ))/2 = (μ+μ)/2 = μ ✓
b — Varyanslar
Var(T₁) = Var(X̄) = σ²/n
Var(T₂) = (Var(X₁)+Var(Xₙ))/4 = (2σ²)/4 = σ²/2
b — Karşılaştırma
n ≥ 3 için: Var(T₁) = σ²/n < σ²/2 = Var(T₂)
→ T₁ = X̄ daha etkindir.
T₂ yalnızca iki gözlem kullandığından, n−2 gözlemin taşıdığı bilgiyi tamamen göz ardı eder. T₁ ise tüm n gözlemi ortalayarak çok daha küçük bir varyansa ulaşır. Örneğin n = 10 için Var(T₁) = σ²/10, Var(T₂) = σ²/2; T₁ beş kat daha hassastır.
NT–10
İki Parametreli Kaydirilmiş Üstel — EÇO ve Yansızlık
EÇO · İki Parametre · İleri İleri
▾
X₁, …, Xₙ b.ö.d. aşağıdaki iki parametreli kaydirilmiş üstel dağılımdan gelmektedir:

f(x; θ, μ) = (1/θ) e−(x−μ)/θ,   x ≥ μ,   θ > 0,   −∞ < μ < ∞
a)μ ve θ için EÇO tahmin edicilerini türetiniz. (İpucu: önce μ için olabilirlik fonksiyonunun μ’ya göre nasıl davrandığını inceleyin.)
b)μ̂’nın yansız olup olmadığını gösteriniz. Yansız düzeltmeyi yazınız.
Adım Adım Çözüm
Olabilirlik
L(θ,μ) = ∏ni=1 (1/θ) e−(xᵢ−μ)/θ = (1/θn) exp[−(1/θ)∑(xᵢ−μ)]
Koşul: μ ≤ x(1) = min(xᵢ)
a — μ EÇO
∑(xᵢ−μ) = ∑xᵢ − nμ: μ arttıkça ∑(xᵢ−μ) azalır, exp[−·/θ] artar ve L artar.
μ’nun izin verilen en büyük değeri x(1) olduğundan:
μ̂ = X(1) = min(X₁, …, Xₙ)
a — Log-olabilirlik (μ sabit)
ℓ(θ | μ̂) = −n lnθ − (1/θ)∑(xᵢ−x(1))
a — Türev & θ EÇO
dℓ/dθ = −n/θ + (1/θ²)∑(xᵢ−x(1)) = 0
θ̂ = ∑(Xᵢ−X(1))/n = X̄ − X(1)
b — E(X(1))
X(1)’in dağılımı kaydirilmiş üsteldir: E(X(1)) = μ + θ/n
E(μ̂) = μ + θ/n ≠ μ → yansız değil.
Yansız düzeltme: μ̂yansız = X(1) − θ̂/n
NT–11
Beta Benzeri Dağılım — EÇO ve Momentler
EÇO · Momentler · İleri İleri
▾
X₁, …, Xₙ b.ö.d. aşağıdaki dağılımdan gelmektedir:

f(x; θ) = θ xθ−1,   0 < x < 1,   θ > 0
a)EÇO yöntemiyle θ tahmin edicisini türetiniz.
b)Momentler yöntemiyle θ tahmin edicisini türetiniz.
Adım Adım Çözüm
a — Olabilirlik
L(θ) = ∏ni=1 θ xᵢθ−1 = θn ∏xᵢθ−1
a — Log-olabilirlik
ℓ(θ) = n lnθ + (θ−1)∑ln(xᵢ)
a — Türev & çozüm
dℓ/dθ = n/θ + ∑ln(xᵢ) = 0
θ̂ᴥ = −n/∑ln(Xᵢ)
b — 1. moment
μ₁ = E(X) = ∫01 x · θxθ−1dx = θ · [xθ+1/(θ+1)]01 = θ/(θ+1)
m₁ = X̄ eşitle: X̄(θ+1) = θ
θ̂ᴩ = X̄/(1−X̄)
NT–12
Poisson — Momentler, EÇO, Yansızlık ve Tutarlılık
Tüm Özellikler · Poisson Orta
▾
X₁, …, Xₙ b.ö.d. Poisson(λ) dağılımından gelen bir örneklemde ∑xᵢ = 48, n = 16 gözlemlenmiştir.
a)λ için hem momentler hem EÇO tahmin edicisini yazınız ve sayısal değerini hesaplayınız.
b)Bu tahmin edicinin yansız, tutarlı ve etkin olduğunu gösteriniz. (Poisson için I(λ) = n/λ.)
Adım Adım Çözüm
a
Her iki yöntem de λ̂ = X̄ verir.
λ̂ = 48/16 = 3.0
b — Yansızlık
E(X̄) = λ ✓
b — Tutarlılık
Var(X̄) = λ/n → 0  &&  Bias = 0 → tutarlı ✓
b — Etkinlik
ln f(x;λ) = x lnλ − λ − ln(x!)
Form 1: ∂ln f/∂λ = x/λ − 1  →  E[(∂ln f/∂λ)²] = Var(X/λ) = Var(X)/λ² = 1/λ
I(λ) = n · E[(∂ln f/∂λ)²] = n/λ
Form 2: ∂²ln f/∂λ² = −x/λ²  →  −E[∂²ln f/∂λ²] = E(X)/λ² = λ/λ² = 1/λ
I(λ) = −nE[∂²ln f/∂λ²] = n/λ ✓
Cramér-Rao alt sınırı: 1/I(λ) = λ/n
Var(X̄) = λ/n = alt sınır  →  X̄ etkindir ✓
📋 Formül Başvuru Kartları — Nokta Tahmini
EÇO — En Çok Olabilirlik
L(θ) = ∏i f(xᵢ; θ)
ℓ(θ) = ln L(θ) = ∑ ln f(xᵢ; θ)
dℓ/dθ = 0 → θ̂ᴥ
d²ℓ/dθ² < 0 doğrulaması
Momentler Yöntemi
μk = E(Xk)  (teorik moment)
mk = ∑Xᵢk/n  (örneklem momenti)
μk(θ) = mk → θ çöz
Tahmin Edici Özellikleri
Yansızlık: E(T) = θ
Bias: b(T) = E(T) − θ
Etkinlik: Var(T₁) < Var(T₂)
Tutarlılık: Tₙ →p θ
Cramér-Rao Alt Sınırı
Fisher bilgi miktarı — iki eşdeğer form:
I(θ) = nE⁰[(∂ln fθ(x)/∂θ)²]
I(θ) = −nE[∂²ln fθ(x)/∂θ²]
─────────────────
Var(T̂) ≥ 1/I(θ)
