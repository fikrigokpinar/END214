NT–01
Üstel Dağılım — EÇO Tahmini
EÇO · Üstel Temel
▾
X₁, X₂, …, Xₙ bağımsız ve özdeş dağılımlı (b.ö.d.) rasgele değişkenler, parametresi θ > 0 olan üstel dağılımdan gelmektedir:

f(x; θ) = θ e−θx,   x > 0
a)θ parametresinin en çok olabilirlik tahmin edicisini (EÇO) türetiniz.
b)EÇO tahmin edicisinin yansız olup olmadığını inceleyiniz.
Adım Adım Çözüm
Log-olas.
L(θ) = ∏i θ e−θxᵢ = θn e−θ∑xᵢ
ℓ(θ) = n lnθ − θ∑xᵢ
Türev
dℓ/dθ = n/θ − ∑xᵢ = 0
θ = n/∑xᵢ = 1/X̄
a
EÇO: θ̂ = 1/X̄ = n/∑Xᵢ
İkinci türev d²ℓ/dθ² = −n/θ² < 0 → maksimum ✓
b
E(X̄) = 1/θ olduğundan E(1/X̄) ≠ θ genel olarak (Jensen eşitsizliği, 1/x dışbükey).
→ θ̂ = 1/X̄ yansız değildir.
Öte yandan (n−1)/(∑Xᵢ) yansız tahmin edici olarak gösterilebilir.
NT–02
Poisson Dağılımı — EÇO Tahmini ve Yansızlık
EÇO · Poisson Temel
▾
X₁, …, Xₙ b.ö.d. Poisson(λ) dağılımından gelmektedir:

P(X = x; λ) = e−λ λx / x!,   x = 0,1,2,…
a)λ parametresinin EÇO'sunu türetiniz.
b)EÇO tahmin edicisinin yansız olduğunu gösteriniz.
Adım Adım Çözüm
Log-olas.
ℓ(λ) = ∑[xᵢ lnλ − λ − ln(xᵢ!)]
= (∑xᵢ) lnλ − nλ − sabit
Türev
dℓ/dλ = ∑xᵢ/λ − n = 0  →  λ̂ = X̄ = ∑Xᵢ/n
b
E(λ̂) = E(X̄) = E(X₁) = λ  [Poisson'da E(X) = λ]
→ X̄ yansızdır.
NT–03
Düzgün Dağılım — Momentler Yöntemi
Momentler · Düzgün Temel
▾
X₁, …, Xₙ b.ö.d. Düzgün(0, θ) dağılımından gelmektedir:

f(x; θ) = 1/θ,   0 < x < θ
a)Momentler yöntemiyle θ için bir tahmin edici türetiniz.
b)Bu tahmin edicinin yansız olup olmadığını gösteriniz.
Adım Adım Çözüm
a — 1. moment
μ₁ = E(X) = θ/2
Örneklem 1. momenti: m₁ = X̄
θ/2 = X̄  →  θ̂ᴩ = 2X̄
b
E(2X̄) = 2·E(X̄) = 2·θ/2 = θ
→ 2X̄ yansızdır.
NT–04
Normal Dağılım — EÇO ile μ ve σ² Tahmini
EÇO · Normal Temel
▾
X₁, …, Xₙ b.ö.d. N(μ, σ²) dağılımından gelmektedir.
a)μ ve σ² için EÇO tahmin edicilerini türetiniz.
b)σ̂² = ∑(Xᵢ−X̄)²/n tahmin edicisinin yansız olup olmadığını tartışınız. Yansız hale getiriniz.
Adım Adım Çözüm
Log-olas.
ℓ(μ,σ²) = −n/2 · ln(2πσ²) − ∑(xᵢ−μ)²/(2σ²)
a
∂ℓ/∂μ = ∑(xᵢ−μ)/σ² = 0  →  μ̂ = X̄
∂ℓ/∂σ² = −n/(2σ²) + ∑(xᵢ−μ)²/(2σ⁴) = 0  →  σ̂² = ∑(Xᵢ−X̄)²/n
b
E[∑(Xᵢ−X̄)²] = (n−1)σ² olduğundan:
E(σ̂²) = (n−1)σ²/n ≠ σ²  →  yansız değil.
Yansız tahmin edici: S² = ∑(Xᵢ−X̄)²/(n−1)
NT–05
Düzgün Dağılım — EÇO Tahmini ve Yansızlık Düzeltmesi
EÇO · Düzgün · Yansızlık Orta
▾
X₁, …, Xₙ b.ö.d. Düzgün(0, θ) dağılımından gelmektedir.
a)θ parametresinin EÇO'sunu türetiniz.
b)EÇO tahmin edicisinin yansız olmadığını gösteriniz ve yansız versiyonunu bulunuz.
Adım Adım Çözüm
a — Olabilirlik
L(θ) = (1/θ)n,   θ ≥ X(n) = max(Xᵢ) koşuluyla
L(θ), θ arttıkça azaldığından maksimum θ = X(n)'de sağlanır.
EÇO: θ̂ = X(n) = max(X₁, …, Xₙ)
b — Yansızlık
X(n)'nin yoğunluk fonksiyonu: f(n)(x) = nxn−1/θn,   0 < x < θ
E(X(n)) = ∫0θ x · nxn−1/θn dx = nθ/(n+1)
E(X(n)) = nθ/(n+1) ≠ θ → yansız değil.
Yansız tahmin edici: T = (n+1)/n · X(n)
E(T) = (n+1)/n · nθ/(n+1) = θ ✓
NT–06
Genel Kuvvet Dağılımı — Momentler ve EÇO Karşılaştırması
Momentler · EÇO · Karşılaştırma Orta
▾
X₁, …, Xₙ b.ö.d. aşağıdaki dağılımdan gelmektedir:

f(x; θ) = (θ+1) xθ,   0 < x < 1,   θ > −1
a)Momentler yöntemiyle θ için bir tahmin edici bulunuz.
b)EÇO yöntemiyle θ için bir tahmin edici bulunuz.
c)İki tahmin edicinin ifadelerini karşılaştırarak yorumlayınız.
Adım Adım Çözüm
a — Momentler
μ₁ = E(X) = ∫01 x(θ+1)xθdx = (θ+1)/(θ+2)
m₁ = X̄ eşitle:
(θ+1)/(θ+2) = X̄  →  θ+1 = X̄(θ+2)  →  θ(1−X̄) = 2X̄−1
θ̂ᴩ = (2X̄−1)/(1−X̄)
b — EÇO
ℓ(θ) = n ln(θ+1) + θ ∑ ln(xᵢ)
dℓ/dθ = n/(θ+1) + ∑ ln(xᵢ) = 0
θ̂ᴥ = −1 − n/∑ln(Xᵢ) = −1 − 1/̅(ln X)
c
İki yöntem farklı formüller vermektedir. Momentler X̄'ya dayalıyken EÇO ∑ln(Xᵢ)'e (geometrik ortalamaya) dayalıdır. EÇO genel olarak asimptotik etkindir; bu dağılımda da EÇO tercih edilir.
NT–07
Üstel Dağılım — İki Yöntemin Karşılaştırması ve Yansızlık
EÇO · Momentler · Yansızlık Orta
▾
X₁, …, Xₙ b.ö.d. μ ortalama parametreli üstel dağılımdan gelmektedir:

f(x; μ) = (1/μ) e−x/μ,   x > 0,   μ > 0
a)Momentler yöntemiyle μ için bir tahmin edici bulunuz ve yansızlığını gösteriniz.
b)EÇO yöntemiyle μ için bir tahmin edici bulunuz. İki yöntem aynı sonucu verir mi?
c)X̄'nın tutarlı tahmin edici olduğunu argümanlayınız.
Adım Adım Çözüm
a
μ₁ = E(X) = μ  →  m₁ = X̄  →  μ̂ᴩ = X̄
E(X̄) = μ → yansızdır. ✓
b
ℓ(μ) = −n lnμ − ∑xᵢ/μ
dℓ/dμ = −n/μ + ∑xᵢ/μ² = 0  →  μ̂ᴥ = X̄
İki yöntem aynı sonucu vermektedir.
c
Büyük Sayılar Yasası (BSY) gereği X̄ →p E(X) = μ.
Ayrıca Var(X̄) = μ²/n → 0, n → ∞.
→ X̄ tutarlı tahmin edicidir.
NT–08
İki Parametreli Düzgün Dağılım — Momentler Yöntemi
Momentler · Düzgün · İki Parametre Orta
▾
X₁, …, Xₙ b.ö.d. Düzgün(α, β) dağılımından gelmektedir:

f(x; α, β) = 1/(β−α),   α < x < β
a)Momentler yöntemiyle α ve β için tahmin edicileri türetiniz.
b)Elde edilen tahmin edicilerin yansızlığını tartışınız.
Adım Adım Çözüm
a — Momentler
μ₁ = (α+β)/2  |  μ₂ = E(X²) = (α²+αβ+β²)/3
Var(X) = μ₂−μ₁² = (β−α)²/12
Örneklemden: m₁ = X̄,   m₂ = ∑Xᵢ²/n
Denklem sistemi:
(α+β)/2 = X̄   →   α+β = 2X̄   …(1)
(β−α)²/12 = S²n  →  β−α = 2√(3S²n)  …(2)
Burada S²n = ∑(Xᵢ−X̄)²/n.
β̂ = X̄ + √(3S²n)  |  α̂ = X̄ − √(3S²n)
b
E(X̄) = (α+β)/2 ✓. Ancak S²n = (n−1)S²/n yansız değildir; dolayısıyla α̂ ve β̂ yansız değildir. S² = ∑(Xᵢ−X̄)²/(n−1) kullanılırsa yanlılık azalır.
NT–09
Etkinlik Karşılaştırması — İki Yansız Tahmin Edici
Etkinlik · Karşılaştırma Orta
▾
X₁, …, Xₙ b.ö.d. N(μ, σ²) dağılımından gelmektedir. Aşağıdaki iki tahmin edici önerilmektedir:

T₁ = X̄   |   T₂ = (X₁ + Xₙ) / 2
a)Her iki tahmin edicinin yansız olduğunu gösteriniz.
b)Varyanslılıklarını karşılaştırınız. Hangisi daha etkindir?
c)Göreli etkinlik (RE) değerini hesaplayınız ve yorumlayınız.
Adım Adım Çözüm
a
E(T₁) = E(X̄) = μ ✓
E(T₂) = (E(X₁)+E(Xₙ))/2 = μ ✓
b
Var(T₁) = σ²/n
Var(T₂) = (Var(X₁)+Var(Xₙ))/4 = 2σ²/4 = σ²/2
n ≥ 3 için: σ²/n < σ²/2  →  T₁ = X̄ daha etkindir.
c
RE(T₁, T₂) = Var(T₂)/Var(T₁) = (σ²/2)/(σ²/n) = n/2
n = 10 için RE = 5 → T₂ aynı hassasiyete ulaşmak için 5 kat daha fazla bilgiye ihtiyaç duyar.
NT–10
Tutarlılık — Üstel Dağılımda Formal Kanıt
Tutarlılık · Üstel Orta
▾
X₁, …, Xₙ b.ö.d. Exp(θ) dağılımından gelmektedir (ortalama 1/θ). θ̂ = 1/X̄ tahmin edicisi düşünülmektedir.
a)X̄'nın tutarlı tahmin edici olduğunu iki koşul üzerinden gösteriniz: (i) yanlılık → 0, (ii) varyans → 0.
b)Sürekli dönüşüm (invariance) özelliğini kullanarak θ̂ = 1/X̄'nın da tutarlı olduğunu argümanlayınız.
Adım Adım Çözüm
a — X̄
E(X̄) = 1/θ → yanlılık = 0 ✓
Var(X̄) = Var(X)/n = (1/θ²)/n → 0 (n→∞) ✓
X̄ tutarlıdır.
b
BSY: X̄ →p 1/θ.
g(t) = 1/t sürekli bir fonksiyondur. Sürekli dönüşüm teoremi gereği:
g(X̄) = 1/X̄ →p g(1/θ) = θ.
→ θ̂ = 1/X̄ tutarlıdır.
NT–11
İki Parametreli Üstel — EÇO ve Yansızlık
EÇO · İki Parametre · İleri İleri
▾
X₁, …, Xₙ b.ö.d. aşağıdaki iki parametreli dağılımdan gelmektedir:

f(x; θ, μ) = θ e−θ(x−μ),   x ≥ μ,   θ > 0,   −∞ < μ < ∞
a)μ ve θ için EÇO tahmin edicilerini türetiniz. (İpucu: önce μ için düşünün, olabilirlik μ'ya göre nasıl davranır?)
b)μ̂'nın yansız olup olmadığını araştırınız.
Adım Adım Çözüm
a — μ EÇO
L(θ,μ) = θn exp(−θ∑(xᵢ−μ)) = θn e−θ(∑xᵢ−nμ), koşul: μ ≤ x(1)
∑(xᵢ−μ) = ∑xᵢ − nμ: μ arttıkça L artar → μ'nun en büyük değeri x(1)'dir.
μ̂ = X(1) = min(Xᵢ)
a — θ EÇO
ℓ(θ | μ̂) = n lnθ − θ∑(xᵢ−x(1))
dℓ/dθ = n/θ − ∑(xᵢ−x(1)) = 0
θ̂ = n / ∑(Xᵢ−X(1))
b
X(1)'in dağılımı: Exp(θ) + μ kaydırılmış, yani E(X(1)) = μ + 1/(nθ).
E(μ̂) = μ + 1/(nθ) ≠ μ → yansız değil.
Yansız düzeltme: μ̂yansız = X(1) − 1/(nθ̂)
NT–12
Beta Benzeri Dağılım — İki Yöntemle Karşılaştırma ve Etkinlik
EÇO · Momentler · Etkinlik · İleri İleri
▾
X₁, …, Xₙ b.ö.d. aşağıdaki dağılımdan gelmektedir:

f(x; θ) = θ xθ−1,   0 < x < 1,   θ > 0
a)EÇO ile θ tahmin edicisini bulunuz.
b)Momentler yöntemiyle θ tahmin edicisini bulunuz.
c)Her iki tahmin edicinin asimptotik varyansını karşılaştırarak hangisinin daha etkin olduğunu gösteriniz.
Adım Adım Çözüm
a
ℓ(θ) = n lnθ + (θ−1)∑ln(xᵢ)
dℓ/dθ = n/θ + ∑ln(xᵢ) = 0
θ̂ᴥ = −n/∑ln(Xᵢ) = −1/̅(ln X)
b
E(X) = ∫01 x·θxθ−1dx = θ/(θ+1)
X̄ = θ/(θ+1)  →  X̄(θ+1) = θ  →  θ̂ᴩ = X̄/(1−X̄)
c
Fisher bilgisi: I(θ) = n/θ²  →  Cramér-Rao alt sınırı = θ²/n
Asimptotik Var(θ̂ᴥ) = θ²/n (EÇO Cramér-Rao'ya ulaşır) ✓
Momentler: Var(X̄) = Var(X)/n = [θ/(θ+1)²]/(θ+2) · 1/n (delta yöntemiyle)
EÇO daha etkindir; Cramér-Rao sınırına ulaşır.
NT–13
Normal Dağılım — Tutarlılık ve MSE Karşılaştırması
Tutarlılık · MSE · Normal · İleri İleri
▾
X₁, …, Xₙ b.ö.d. N(μ, σ²) dağılımından gelmektedir. σ² için iki tahmin edici önerilmektedir:

T₁ = S² = ∑(Xᵢ−X̄)²/(n−1)   |   T₂ = ∑(Xᵢ−X̄)²/n
a)Her ikisinin yanlılığını (bias) hesaplayınız.
b)Her ikisinin MSE (Ortalama Karesel Hata) değerini hesaplayınız. (İpucu: Normal için Var(S²) = 2σ⁴/(n−1).)
c)MSE bakımından hangi tahmin edici daha iyidir? Her ikisinin tutarlı olduğunu gösteriniz.
Adım Adım Çözüm
a
E(T₁) = σ²  →  Bias(T₁) = 0 (yansız)
E(T₂) = (n−1)σ²/n  →  Bias(T₂) = −σ²/n
b
MSE(T₁) = Var(T₁) + 0² = Var(S²) = 2σ⁴/(n−1)
T₂ = (n−1)/n · S²  →  Var(T₂) = [(n−1)/n]² · 2σ⁴/(n−1) = 2(n−1)σ⁴/n²
MSE(T₂) = Var(T₂) + [Bias(T₂)]² = 2(n−1)σ⁴/n² + σ⁴/n²
= (2n−1)σ⁴/n²
c
MSE(T₂)/MSE(T₁) = [(2n−1)/n²] / [2/(n−1)] = (2n−1)(n−1)/(2n²)
n ≥ 2 için bu oran < 1  →  T₂ MSE bakımından daha iyidir.
Tutarlılık: Var(T₁) → 0 ve Bias(T₁) = 0 → T₁ tutarlı ✓
Var(T₂) → 0 ve Bias(T₂) = −σ²/n → 0 → T₂ tutarlı ✓
NT–14
Poisson — Momentler, EÇO, Yansızlık ve Tutarlılık
Tüm Özellikler · Poisson Orta
▾
X₁, …, Xₙ b.ö.d. Poisson(λ) dağılımından gelen bir örneklemde ∑xᵢ = 48, n = 16 gözlemlenmiştir.
a)λ için hem momentler hem EÇO tahmin edicisini yazınız ve sayısal değerini hesaplayınız.
b)Bu tahmin edicinin yansız, tutarlı ve etkin olduğunu gösteriniz. (Poisson için I(λ) = n/λ.)
Adım Adım Çözüm
a
Her iki yöntem de λ̂ = X̄ verir.
λ̂ = 48/16 = 3.0
b — Yansızlık
E(X̄) = λ ✓
b — Tutarlılık
Var(X̄) = λ/n → 0  &&  Bias = 0 → tutarlı ✓
b — Etkinlik
Cramér-Rao alt sınırı = 1/I(λ) = λ/n
Var(X̄) = λ/n = alt sınır → etkin (UMVUE) ✓
📋 Formül Başvuru Kartları — Nokta Tahmini
EÇO — En Çok Olabilirlik
L(θ) = ∏i f(xᵢ; θ)
ℓ(θ) = ln L(θ) = ∑ ln f(xᵢ; θ)
dℓ/dθ = 0 → θ̂ᴥ
d²ℓ/dθ² < 0 doğrulaması
Momentler Yöntemi
μk = E(Xk)  (teorik moment)
mk = ∑Xᵢk/n  (örneklem momenti)
μk(θ) = mk → θ çöz
Tahmin Edici Özellikleri
Yansızlık: E(T) = θ
Bias: b(T) = E(T) − θ
Etkinlik: Var(T₁) < Var(T₂)
Tutarlılık: Tₙ →p θ
MSE = Var + Bias²
Cramér-Rao Alt Sınırı
I(θ) = −E[∂²ln f/∂θ²]
Var(T) ≥ 1/I(θ)
EÇO asimptotik olarak bu sınıra ulaşır
UMVUE: alt sınıra ulaşan yansız T
