▲ Arasınav Öncesi
🔑Temel Tanımlar
N, μ, σ² Yığın büyüklüğü, ortalaması, varyansı (parametre)
n, x̄, s² Örneklem büyüklüğü, ortalaması, varyansı (istatistik)
m ≥ √n Çokluk dağılımı sınıf sayısı (m), n: gözlem sayısı
c ≥ L/m Sınıf aralığı; L = maks − min
f/N Nispi (oransal) çokluk
Ölçme düzeyleri Sınıflama → Sıralama → Eşit aralıklı → Oranlama
📊Merkezi Eğilim Ölçüleri
μ = Σxᵢ/N Basit dizi aritmetik ortalama
μ = Σx̄ⱼfⱼ/N Çokluk dağılımı AO; x̄ⱼ: sınıf orta değeri
Me = a(N+1)/2 N tek iken medyan; N çift → iki ortanın ort.
Me = L+(c/f)(N/2−d) Çokluk dağılımı medyan; d: önceki birikimli f
Mo = L+fs/(fs+fo)·c Mod; fo: önceki, fs: sonraki sınıf çokluğu
Simetrik AO = Me = Mo  |  Sağa çarpık: Mo<Me<AO  |  Sola: AO<Me<Mo
📉Merkezi Dağılım Ölçüleri
Açıklık = maks−min Yalnızca iki uç değeri kullanır
Q = (Q₃−Q₁)/2 Çeyrek ayrılış; IQR = Q₃−Q₁
OA = (1/N)Σ|xᵢ−μ| Ortalama ayrılış
σ² = (1/N)Σ(xᵢ−μ)² Varyans (tanımlama)
σ² = (1/N)Σxᵢ²−μ² Varyans (hesaplama); σ = √σ²
DK = σ/μ Değişim katsayısı; DK küçük → homojen
Q₁−1.5·IQR Kutu grafiği alt/üst sınır (aykırı değer eşiği)
📈Grafikler
Histogram Sürekli değişken; çubuklar bitişik; eşit sınıf aralığı
Dengesiz hist. En sık aralığı bul, büyük sınıfları eşit parçalara böl
Çubuk grafiği Kategorik değişken; çubuklar ayrık (boşluk var)
Açı = (f/N)×360° Daire grafiği dilim açısı
Açık uçlu sınıf Histogram ve AO/σ² hesabında işlem dışı tutulur
🎯Nokta Tahmini
L(θ) = Πf(xᵢ;θ) Olabilirlik fonksiyonu; ∂lnL/∂θ = 0 → EÇO
E(Xʳ) = μʳ Momentler yöntemi: örnek momentini yığın momentine eşitle
E(θ̂) = θ Yansızlık; MSE = Var(θ̂) + Bias²
I(θ) = −nE[∂²lnf/∂θ²] Fisher bilgisi (n gözlem için)
Var(θ̂) ≥ 1/I(θ) Cramér–Rao alt sınırı; eşitlikte → etkin tahmin
θ̂ → θ (n→∞) Tutarlılık: n büyüdükçe tahmin θ'ya yaklaşır
▼ Arasınav Sonrası
📐Güven Aralığı
x̄ ± zα/2·σ/√n μ için GA — σ bilinir (z dağılımı)
x̄ ± tα/2,n−1·s/√n μ için GA — σ bilinmez (t, df=n−1)
p ± zα/2√(p(1−p)/n) Oran π için GA; p: örneklem oranı
(x̄₁−x̄₂) ± zα/2·SE İki grup μ farkı için GA — σ bilinir
sp² = ((n₁−1)s₁²+(n₂−1)s₂²)/(n₁+n₂−2) Pooled varyans; df = n₁+n₂−2
n = (zα/2·σ/E)² Gerekli örneklem büyüklüğü; E: hata payı
🔬Hipotez Testi
Z = (x̄−μ₀)/(σ/√n) Tek örneklem z-testi — σ bilinir
T = (x̄−μ₀)/(s/√n) Tek örneklem t-testi — σ bilinmez; df = n−1
Z = (p−π₀)/√(π₀(1−π₀)/n) Oran testi; π: yığın oranı, p: örneklem oranı
T = d̄/(sd/√n) Eşleştirilmiş t; d̄: fark ort., df = n−1
T = (x̄₁−x̄₂)/(sp√(1/n₁+1/n₂)) İki bağımsız grup pooled t; df = n₁+n₂−2
|T|>tkrit → H₀ ret Karar kuralı; tek kuyruk: T>t ya da T<−t
📈Doğrusal Regresyon
β̂₁ = Sxy/Sxx EKK eğim; Sxy=Σxᵢyᵢ−nȳx̄, Sxx=Σxᵢ²−nx̄²
β̂₀ = ȳ − β̂₁x̄ EKK sabit terim
TKT = Σ(yᵢ−ȳ)² Toplam Kareler Toplamı
RKT = β̂₁·Sxy Regresyon KT = Σ(ŷᵢ−ȳ)²
AKT = TKT − RKT Artık Kareler Toplamı = Σ(yᵢ−ŷᵢ)²
R² = RKT/TKT Belirlilik katsayısı; 0 ≤ R² ≤ 1
