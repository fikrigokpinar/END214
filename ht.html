HT–01
Tek Örneklemde Ortalama Testi (z-testi, σ bilinir)
z-testi Temel
▾
Bir içecek şirketinin otomatik dolum makinesi her şişeye 500 ml doldurmaktadır. Kalite kontrol müdürü makinenin doğru çalışıp çalışmadığını kontrol etmek istemektedir. Rastgele seçilen n = 49 şişenin ortalama dolumu x̄ = 497 ml olarak ölçülmüştür. Yığın standart sapması σ = 14 ml bilinmektedir.

%5 anlamlılık düzeyinde makinenin doğru çalışıp çalışmadığını test ediniz.
Adım Adım Çözüm
H
H₀: μ = 500  |  H₁: μ ≠ 500  (çift kuyruklu)
α
α=0.05 → Kritik değer: ±z0.025 = ±1.96
z
z = (x̄−μ₀)/(σ/√n) = (497−500)/(14/7) = −3/2 = −1.50
Karar
|z|=1.50 < 1.96 → Ret bölgesinde değil
H₀ REDDEDİLEMEZ
%5 anlamlılık düzeyinde makinenin 500 ml doldurduğu hipotezi reddedilemez.
HT–02
Bağımlı Örneklemde Hipotez Testi (Eşleştirilmiş t-testi)
Eşleştirilmiş t Orta
▾
Bir fast-food zinciri, yeni menü düzenlemesinin sipariş hazırlama süresini kısalttığını iddia etmektedir. 6 farklı şubede eski ve yeni menüde sipariş hazırlama süreleri (saniye) ölçülmüştür:
Şube123456
Eski Menü280310340380420360
Yeni Menü240270290310350300
%5 anlamlılık düzeyinde şirketin iddiasını test ediniz.
Adım Adım Çözüm
H
H₀: μD ≤ 0  |  H₁: μD > 0  (tek kuyruklu sağ)  [d = eski−yeni]
d
d = [40, 40, 50, 70, 70, 60] → d̄=55
sd²=[Σd²−n·d̄²]/(n−1)=[19400−18150]/5=250 → sd=15.81
t
t = d̄/(sd/√n) = 55/(15.81/√6) = 55/6.45 = 8.53
Karar
df=5, t0.05,5=2.015 (tek kuyruk)
t=8.53 > 2.015 → Ret bölgesinde
H₀ REDDEDİLİR
Yeni menü hazırlama süresini anlamlı ölçüde kısaltmıştır.

Hat 1401250120

%5 anlamlılık düzeyinde iki hattın ortalama ömürlerinin eşit olup olmadığını test ediniz.
Adım Adım Çözüm
H

HT–04
Tek Örneklemde Oran Testi (z-testi, π için)
Oran testi · π Temel
▾
Bir anket şirketi, internet alışverişinin toplumda %40'ın üzerinde yaygınlaştığını öne sürmektedir. Yapılan bir araştırmada n = 200 kişiden 90 tanesi düzenli olarak internet alışverişi yaptığını belirtmiştir.

%5 anlamlılık düzeyinde bu iddiayı test ediniz.
Adım Adım Çözüm
H
H₀: π ≤ 0.40  |  H₁: π > 0.40  (sağ kuyruklu)
p
p = 90/200 = 0.45  (örneklem oranı)
z
z = (p−π₀)/√(π₀(1−π₀)/n) = (0.45−0.40)/√(0.40·0.60/200) = 0.05/0.03464 = 1.443
Karar
z0.05=1.645 (tek kuyruk)
1.443 < 1.645 → Ret bölgesinde değil
H₀ REDDEDİLEMEZ
İnternet alışverişinin %40'ın üzerinde olduğu istatistiksel olarak desteklenmemektedir.
HT–05
İki Bağımsız Örneklemde Ortalama Farkı Testi — Pooled t
İki örneklem · pooled t Orta
▾
İki farklı diyet programının katılımcılara bir ay sonunda verdirdiği kilo kaybı (kg) karşılaştırılmaktadır. Yığın varyanslarının eşit olduğu varsayılmaktadır.
Programnx̄ (kg)s (kg)
A125.81.4
B104.61.6
%5 anlamlılık düzeyinde iki programın ortalama kilo kaybının eşit olup olmadığını test ediniz.
Adım Adım Çözüm
H
H₀: μ₁=μ₂  |  H₁: μ₁≠μ₂  (çift kuyruklu)
sp
sp² = [(n₁−1)s₁²+(n₂−1)s₂²]/(n₁+n₂−2)
= [11·1.96 + 9·2.56]/20 = [21.56+23.04]/20 = 44.6/20 = 2.23
sp = 1.493
t
t = (x̄₁−x̄₂) / (sp·√(1/n₁+1/n₂)) = (5.8−4.6) / (1.493·√(1/12+1/10))
= 1.2 / (1.493·0.4282) = 1.2/0.6393 = 1.877
Karar
df=20, t0.025,20=2.086 (çift kuyruk)
|t|=1.877 < 2.086 → Ret bölgesinde değil
H₀ REDDEDİLEMEZ
%5 anlamlılık düzeyinde iki programın ortalama kilo kaybı arasında anlamlı bir fark bulunamamıştır.
📋 Formül Başvuru Kartları — Hipotez Testi
Genel Test Akışı
1. H₀ ve H₁'i kur
2. α ve kritik değeri belirle
3. Test istatistiğini hesapla
4. Karar: |T| > t_krit → Reddet
Tek Örneklem — Ortalama
σ biliniyor: z = (x̄−μ₀)/(σ/√n)
σ bilinmiyor: t = (x̄−μ₀)/(s/√n)
df = n−1 (t için)
Eşleştirilmiş t-Testi
di=x₁ᵢ−x₂ᵢ
t = d̄/(sd/√n)
df = n−1
İki Bağımsız Ortalama (σ biliniyor)
z = (x̄₁−x̄₂)/√(σ₁²/n₁+σ₂²/n₂)
İki Bağımsız Ortalama — Pooled t
sp² = [(n₁−1)s₁²+(n₂−1)s₂²]/(n₁+n₂−2)
t = (x̄₁−x̄₂)/(sp·√(1/n₁+1/n₂))
df = n₁+n₂−2

z = (p−π₀)/√(π₀(1−π₀)/n)
